Research

무엇을 연구하나요?

"사람이 AI를 만드는 시대"에서 "AI가 AI를 만드는 시대"로. AutoML Lab은 기계학습의 설계 자체를 자동화합니다.

🧬

Neural Architecture Search

신경망 구조 탐색

신경망 설계를 사람이 아닌 알고리즘이 하도록 만듭니다. 밤새 하이퍼파라미터를 손으로 튜닝하는 대신, 탐색 알고리즘이 최적 구조를 찾아냅니다.

⚡

Efficient Deep Learning

효율적 딥러닝

거대한 모델을 스마트폰에서도 돌아가게. 성능은 지키고 크기와 연산량은 줄이는, 작지만 강한 모델을 만드는 기술입니다.

🎯

Feature Selection

특징 선택

수천 개 변수 중 진짜 중요한 것만 골라냅니다. 데이터의 본질을 찾아 모델을 가볍고 정확하게 만드는 연구입니다.

🌱

Evolutionary Algorithm

진화 알고리즘

생물 진화의 원리 — 선택, 교배, 변이 — 로 최적해를 탐색합니다. 자연에서 배운 알고리즘으로 어려운 최적화 문제를 풉니다.
